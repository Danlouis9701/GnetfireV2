Foglight

Gestiona la experiencia del usuario desde varias perspectivas y le permite administrar SLA de rendimiento mediante la detección, el aislamiento y la resolución de problemas de tiempo de respuesta y la captura y reproducción de interacciones reales con el usuario para entender cómo las afectan el diseño y la configuración de aplicaciones. Además, controla y analiza conversiones de transacciones y metadatos de sesiones, descubre patrones de compra/preferencias y determina por qué los usuarios abandonaron transacciones.

Migración hacia Office 365

On Demand Migration for Email

On Demand Migration for Email le permite migrar de forma segura usuarios y datos a plataformas de Office 365, Exchange en las instalaciones o Exchange alojado sin tener que instalar o mantener el software de migración.

Con la capacidad de migrar de manera simultánea múltiples usuarios desde un única consola mientras mueve de forma segura el correo electrónico, los calendarios y las carpetas, podrá lograr lo siguiente:

  • Filtrar rápidamente datos no deseados para reducir plazos de migración.
  • Evitar la pérdida de datos con la preservación automatizada de adjuntos de gran tamaño.
  • Monitorear todo el progreso de la migración con informes de progreso para garantizar la seguridad de sus datos

Enterprise Reporter

Es una solución escalable para la auditoría, el análisis y la presentación de informes sobre la configuración de Microsoft, Active Directory, Windows Server y SQL Server en toda la empresa.

Herramientas para informes predefinidas y customizables que permiten realizar facilmente evaluaciones de cumplimiento y de seguridad, análisis previos y posteriores a la migración, entre otras que ayuda a la planificación estratégica de TI y gestión proactiva para las redes basadas en Microsoft Window.

  • Visibilidad Cumplimiento y seguridad.
  • Evaluación de accesos en tiempo real.
  • Análisis previos y posteriores a migraciones.
  • Revisión de cambios.
  • Recopilación de datos escalables.
  • Almacenamiento eficiente.
  • Reportes automatizados de flujo de trabajo.
  • Separación de funciones.
  • Reportes customizables.
